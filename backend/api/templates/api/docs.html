REST API Endpoints

GET /api/v1/pipelines/

List all accessible pipelines with pagination and filtering.

Query Parameters:

ParameterTypeDescription
pageintegerPage number
page_sizeintegerItems per page (max 1000)
searchstringSearch term
pipeline_typestringFilter by type

WebSocket API

Real-time communication through WebSocket connections with SSE-style messaging.

Connection

const ws = new WebSocket('{{ websocket_api.endpoints.sse }}?token=YOUR_JWT_TOKEN');

ws.onopen = function() {
    console.log('Connected to WebSocket');
};

ws.onmessage = function(event) {
    const data = JSON.parse(event.data);
    console.log('Received:', data);
};

Subscribing to Updates

// Subscribe to record updates
ws.send(JSON.stringify({
    type: 'subscribe',
    subscription: 'record_updates',
    id: 'my_subscription',
    filters: {
        pipeline_id: 'your-pipeline-id'
    }
}));

Authentication

JWT Token Authentication

curl -H "Authorization: Bearer YOUR_JWT_TOKEN" \
     {{ rest_api.base_url }}pipelines/

API Key Authentication

curl -H "X-API-Key: YOUR_API_KEY" \
     {{ rest_api.base_url }}pipelines/

WebSocket Authentication

Include JWT token in query parameter or Authorization header:

// Query parameter
const ws = new WebSocket('wss://api.example.com/ws/sse/?token=YOUR_JWT_TOKEN');

// Authorization header (if supported by client)
const ws = new WebSocket('wss://api.example.com/ws/sse/', [], {
    headers: {
        'Authorization': 'Bearer YOUR_JWT_TOKEN'
    }
});

Code Examples

Python Example

import requests

# Set up authentication
headers = {'Authorization': 'Bearer YOUR_JWT_TOKEN'}
base_url = '{{ rest_api.base_url }}'

# Create a pipeline
pipeline_data = {
    'name': 'Sales Pipeline',
    'description': 'Track sales opportunities',
    'pipeline_type': 'crm'
}
response = requests.post(f'{base_url}pipelines/', 
                        json=pipeline_data, headers=headers)
pipeline = response.json()

# Create a record
record_data = {
    'data': {
        'company_name': 'Acme Corp',
        'contact_email': 'john@acme.com',
        'deal_value': 50000
    }
}
response = requests.post(f'{base_url}pipelines/{pipeline["id"]}/records/',
                        json=record_data, headers=headers)
record = response.json()

JavaScript Example

// Using fetch API
const apiCall = async (endpoint, options = {}) => {
    const response = await fetch(`{{ rest_api.base_url }}${endpoint}`, {
        headers: {
            'Authorization': 'Bearer YOUR_JWT_TOKEN',
            'Content-Type': 'application/json',
            ...options.headers
        },
        ...options
    });
    return response.json();
};

// Create pipeline and record
const pipeline = await apiCall('pipelines/', {
    method: 'POST',
    body: JSON.stringify({
        name: 'Sales Pipeline',
        pipeline_type: 'crm'
    })
});

const record = await apiCall(`pipelines/${pipeline.id}/records/`, {
    method: 'POST',
    body: JSON.stringify({
        data: {
            company_name: 'Acme Corp',
            deal_value: 50000
        }
    })
});
